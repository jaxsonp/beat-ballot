### `/create-playlist`

Creates a new playlist with the specified name

Params:

-   `name`: Name of the playlist to be created

### `/add-user-to-playlist`

Adds a user to a playlist group

Params:

-   `username`: Name of the user
-   `playlist_id`: ID of the playlist
